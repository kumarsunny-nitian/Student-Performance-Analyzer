# Student Performance Analyzer

## Overview

A Machine Learning project that analyzes student performance and predicts outcomes using multiple algorithms.

## Algorithms Used

1. Linear Regression
2. Logistic Regression
3. K-Nearest Neighbors (KNN)
4. K-Means Clustering

## Features

- Marks Prediction
- Placement Prediction
- Student Segmentation
- Data Visualization
- Performance Analysis

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results

- Predicted student marks
- Predicted placement status
- Clustered students into performance groups
